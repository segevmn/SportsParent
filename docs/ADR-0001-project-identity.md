# ADR-0001 — Project Identity & Scope

Status: Accepted · Date: TODAY

- Name: SportsParent — POC (Backend-only)
- Scope (POC): Auth (JWT Simple), RBAC baseline (user/admin), Error format & codes (POC minimal), Auth API contracts (`/auth/signup|login|logout|me`).
- Planned (post-POC): Player Profiles, Video attachments, Scout Search, Opportunities, Teams, Events.
- Out of Scope (POC): AI/advanced video analysis, public frontend, refresh flow, and all non-auth features listed under “Planned”.
